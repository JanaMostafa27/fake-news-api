# Fake News Detection API

Flask-based REST API for fake news detection and news category classification.
Deployed on Railway.

## Project Structure

```
deployment/
├── app.py                  ← Flask application (main file)
├── requirements.txt        ← Python dependencies
├── Procfile                ← Railway/Gunicorn start command
├── .gitignore
├── README.md
└── model_artifacts/        ← Generated by notebook (NOT committed to git)
    ├── multi_task_distilbert.pt
    └── preprocessor.pkl
```

## Endpoint

### POST /predict
Classify a news article.

**Request**
```json
{
  "text": "some news content"
}
```

**Response**
```json
{
  "fake_or_true": "Fake",
  "category": "Politics"
}
```

**Headers required**
```
x-api-key: your_secret_key
Content-Type: application/json
```

### GET /
Health check — no auth required.
